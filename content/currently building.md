# Template for Building a Product
### 1. Business
- Market Research (Competitions and Product market)
- Profit Model
- Partnership Opportunities
- Engagement Potential
### 2. Branding
- Mission Statements
- Building an Engagement Funnel (Reddit/Twitter/TikTok influence)
- Domain name/names
- Blogging/Vlogging
- Content Creation (ads)
- Network Feelers
### 3. Product
- UI/Design + User testing
- Directory/Github (the project skeleton, the Frontend outline)
	- Pages and Layouts
	- Page Navigation
- Core Backend Feature
- Userbase (and how they interact with the backend)
- Data (RDB, Document-Tables etc.)
- Architecture (Lambda, DynamoDB, etc.)
- Deployment (Amplify)
- CTAs (calls to action, ensure marketed feature is a part of app)
- Full Frontend
	- Fonts and Images
	- CSS Styling Template
	- Database fetching
	- Rendering
	- Data mutation
	- Error pages
	- Authentication
	- Metadata
- Tests
- Error Monitoring
### 4. Logistics
- Wiki/Documentation
- Team Structure
- Communications
- Accounts
- Password Management
- Funding Logistics
- Incorporation and Legal stuff
