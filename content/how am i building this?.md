I use Obsidian to stylize .md files and use [Quartz](https://quartz.jzhao.xyz) to turn it into HTML, CSS, and JS. It's all deployed through GH pages

I can test it using 
`npx quartz build --serve`

I can deploy it with 
`npx quartz sync`
