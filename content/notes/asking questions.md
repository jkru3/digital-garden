you need to be able to ask to take agency of your life. But you have to be careful. You can end up asking questions poorly, and while it is a cost of doing business, it’s never fun to be the bad-question-asker, so it helps to give yourself a set of checks before letting your mouth run

So before asking anything, make sure of the following things:
1. is what I want to ask within my **paygrade**?
	1. You are not asking anything that is directorial 
2. Given the context in which a question is asked, the answer is **unclear**
	1. It’s not in the presentation
	2. It wasn’t just talked about
3. The question is **relevant** to the current topic
4. It does not raise any **controversial** points or discussions
5. It is not a **self-aggrandizing** ask
	1. Look what I know!
